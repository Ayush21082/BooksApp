# BooksApp
Pods for BooksApp

## Requirements

- iOS 14.0+
- Swift 5

## Installation
``` ruby
pod 'BookAppsLib'
```

## Usage

Example

  ``` swift
 import SwiftUI
 import BookAppsAPI

struct ContentView: View {
    var body: some View {
       BookAppsAPI()
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}
  ```
