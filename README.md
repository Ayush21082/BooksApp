# BooksApp
Pods for BooksApp

## Requirements

- iOS 14.0+
- Swift 5

## Installation
``` ruby
pod 'BookAppsLib'
```

## Usage

* Run Book App Demo for the result

Example

  ``` swift
 import SwiftUI
 import BookApps

struct ContentView: View {
    var body: some View {
       BookAppsAPI()
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}
  ```
